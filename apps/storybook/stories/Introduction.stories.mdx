# Welcome to @inari/components

**@inari/components** is a component library developed with Vue 3 & Vite using the Inari Design System.

This webpage is a Storybook where you can see all the library components and interact with all their posibilities.

### Features of the component library
- Vue 3
- Vite
- ESLint
- Prettier
- Testing with Vitest
- Storybook
- Documentation

### Library installation

You can install the library from the Inari NPM private repository.

```
yarn add @inari/components
```

### Importing Components

To import the `@inari/components` you need to do the following:

```
import { IclComponent1, IclComponent2 } from "@inari/components";
```

Also you have to import library CSS styles:

```
import "@inari/components/dist/style.css";
```

Importing automatically each component styles without importing global style file is a pending feature.
